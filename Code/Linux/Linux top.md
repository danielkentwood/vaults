In the Linux command line, you can monitor how much resources are available on the computer and track your code running by using the program `top`

`top` : monitor system resources

`top -u $UID` : view only your own processes

When in top, press `m` to change the memory usage to a bar chart which makes it easy to see what % of memory is in use. Press `e` to change memory display units for processes to gigabytes for ease of understanding. Press `shift-w` to save your top display settings so it will load with your preferred settings next time. `shift-m` switches process order to be ordered by memory
